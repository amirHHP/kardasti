Engineered for Candidates, Built for Privacy

A professional workflow that fits right into your browser without bloating your process or leaking your data.

One-Click Selection Flow

No copy-pasting back and forth. Highlight any job posting on LinkedIn, Indeed, or any career portal, right-click, and let Kardasti compose your draft instantly in the background.

Absolute Local Privacy

Your resume, personal details, and API keys are stored solely in your Chrome local browser storage. We have no external backend servers, meaning your confidential career history remains 100% yours.

Multi-LLM Agnostic

Connect using your own API credentials. Kardasti natively detects Google Gemini, OpenAI, Groq, or OpenRouter setups, enabling you to use cost-effective models like `gpt-4o-mini` or `gemini-1.5-flash` directly.

Instant Clipboard & PDF

When generation finishes, the layout engine copies the formatted text to your clipboard and compiles a clean, professional, WinAnsi Helvetica PDF document downloaded directly to your computer.

Frequently Asked Questions

Clear answers to details about privacy, models, and usage.

Where is my resume text sent?

Your resume text is stored locally within Chrome's sandboxed storage. It is only sent directly to the AI provider endpoint (OpenAI, Google, Groq, or OpenRouter) that you configure when you explicitly trigger the "Generate Cover Letter" context menu action. We do not operate server portals or analytics databases, ensuring absolute confidentiality.

Do I need to pay to use Kardasti?

No, the Chrome extension itself is entirely free and open-source. You only need your own API key from an LLM provider. Since modern API calls are extremely cost-effective (e.g., using GPT-4o-mini costs less than $0.002 per letter), you only pay fractions of a cent directly to OpenAI or Google for the API usage.

How does the extension auto-detect Gemini?

Kardasti analyzes the pattern of your API key. If the key begins with the standard Google developer key prefix (AIza), the popup automatically populates the base URL to point to Google's Gemini API endpoints and loads the available Gemini model families (like gemini-1.5-flash).

How is the PDF generated?

The PDF is compiled in-memory inside the Chrome extension background worker using a lightweight, pure JavaScript PDF layout engine. It outputs standard PDF 1.4 objects without referencing heavy external UI scripts, formatting the Helvetica fonts and alignments dynamically to trigger an automatic web download.
